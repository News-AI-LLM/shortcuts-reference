# ⌨ ショートカットキーまとめ

## 📊 Excel

### 基本操作
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + C` | コピー |
| `Ctrl + X` | 切り取り |
| `Ctrl + V` | 貼り付け |
| `Ctrl + Z` | 元に戻す |
| `Ctrl + Y` | やり直し |
| `Ctrl + S` | 上書き保存 |
| `F12` | 名前を付けて保存 |
| `Ctrl + N` | 新しいブックを作成 |
| `Ctrl + O` | ブックを開く |
| `Ctrl + P` | 印刷 |
| `Ctrl + W` | ブックを閉じる |
| `Ctrl + F4` | ブックを閉じる（終了） |

### セル操作
| ショートカット | 動作 |
|--------------|------|
| `F2` | セル編集モード |
| `Esc` | 編集をキャンセル |
| `Enter` | 下のセルに移動／確定 |
| `Tab` | 右のセルに移動 |
| `Shift + Enter` | 上のセルに移動 |
| `Shift + Tab` | 左のセルに移動 |
| `Ctrl + D` | 上からコピー（下方向へ） |
| `Ctrl + R` | 左からコピー（右方向へ） |
| `Ctrl + Enter` | 選択セルに一括入力 |
| `Ctrl + Delete` | カーソル右を削除 |
| `Alt + ↓` | ドロップダウンリスト表示 |

### 選択・移動
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + ←↑→↓` | データ末尾にジャンプ |
| `Ctrl + Shift + ←↑→↓` | データ範囲を選択 |
| `Ctrl + Space` | 列全体を選択 |
| `Shift + Space` | 行全体を選択 |
| `Ctrl + A` | 全選択（範囲→全体） |
| `Ctrl + G / F5` | ジャンプ（セル番地指定） |
| `Ctrl + [` | 参照元セルにジャンプ |
| `Ctrl + ]` | 参照先セルにジャンプ |
| `Ctrl + .` | 選択範囲の角を移動 |
| `Alt + ;` | 可視セルのみ選択 |
| `Ctrl + /` | 配列のセル範囲を選択 |
| `Ctrl + \` | 行内の差異を選択 |
| `Ctrl + Shift + \|` | 列内の差異を選択 |

| `Ctrl + Shift + %` | パーセント書式 |

| `Ctrl + Shift + ~` | 標準書式（初期化） |

| `Ctrl + ;` | 今日の日付を入力 |

### 行・列操作

### 数式・関数

| `Ctrl + F3` | 名前の定義 |
| `Ctrl + Shift + F3` | 選択範囲から名前作成 |

### フィルター・並べ替え

### ピボット・グラフ

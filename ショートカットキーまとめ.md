# ⌨ ショートカットキーまとめ

Excel／Outlook／Windows の厳選ショートカット一覧

---

## 📊 Excel

### 基本操作
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + C` | コピー |
| `Ctrl + X` | 切り取り |
| `Ctrl + V` | 貼り付け |
| `Ctrl + Z` | 元に戻す |
| `Ctrl + Y` | やり直し |
| `Ctrl + S` | 上書き保存 |
| `F12` | 名前を付けて保存 |
| `Ctrl + W` | ブックを閉じる |

### セル操作
| ショートカット | 動作 |
|--------------|------|
| `F2` | セル編集モード |
| `Esc` | 編集をキャンセル |
| `Enter` | 下のセルに移動 |
| `Tab` | 右のセルに移動 |
| `Shift + Enter` | 上のセルに移動 |
| `Shift + Tab` | 左のセルに移動 |
| `Ctrl + D` | 上からコピー（下方向へ） |
| `Ctrl + R` | 左からコピー（右方向へ） |

### 選択・移動
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + ←↑→↓` | データ末尾にジャンプ |
| `Ctrl + Shift + ←↑→↓` | データ範囲を選択 |
| `Ctrl + Space` | 列全体を選択 |
| `Shift + Space` | 行全体を選択 |
| `Ctrl + A` | 全選択（範囲→全体） |
| `Ctrl + G` | ジャンプ（セル番地指定） |
| `Ctrl + [` | 参照元セルにジャンプ |
| `Ctrl + ]` | 参照先セルにジャンプ |

### 書式設定
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + 1` | セルの書式設定ダイアログ |
| `Ctrl + B` | 太字 |
| `Ctrl + I` | 斜体 |
| `Ctrl + U` | 下線 |
| `Ctrl + 5` | 取り消し線 |
| `Ctrl + Shift + ~` | 標準書式（初期化） |
| `Alt + Enter` | セル内改行 |
| `Ctrl + ;` | 今日の日付を入力 |
| `Ctrl + Shift + ;` | 現在時刻を入力 |

### 行・列操作
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + +` | セル/行/列を挿入 |
| `Ctrl + -` | セル/行/列を削除 |
| `Alt → H → O → I` | 列幅を自動調整 |
| `Alt → H → O → A` | 行の高さを自動調整 |
| `Alt → H → W` | 折り返して全体を表示 |

### 数式・関数
| ショートカット | 動作 |
|--------------|------|
| `Alt + =` | SUM（オートSUM） |
| `F4` | 絶対参照に切り替え（$A$1） |
| `Ctrl + \`` | 数式表示/非表示切替 |
| `F9` | 選択範囲を再計算 |
| `Shift + F3` | 関数の挿入ダイアログ |
| `Ctrl + Shift + U` | 数式バーの展開/折りたたみ |

### フィルター・並べ替え
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + Shift + L` | フィルターのオン/オフ |
| `Alt + ↓` | フィルターリストを開く |
| `Ctrl + T` | テーブル作成 |

### ピボット・グラフ
| ショートカット | 動作 |
|--------------|------|
| `Alt + F1` | 標準グラフを即作成 |
| `F11` | グラフシートを作成 |
| `Alt + F5` | ピボットテーブル更新 |

### シート・ブック操作
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + PgDn` | 次のシートへ |
| `Ctrl + PgUp` | 前のシートへ |
| `Shift + F11` | 新しいシートを挿入 |
| `Ctrl + Tab` | 開いているブックを切替 |

### 検索・置換
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + F` | 検索 |
| `Ctrl + H` | 置換 |

## 📧 Outlook

### 基本操作
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + N` | 新しいアイテム作成 |
| `Ctrl + Shift + M` | 新しいメール |
| `Ctrl + Shift + A` | 新しい予定 |
| `Ctrl + Shift + C` | 新しい連絡先 |
| `Ctrl + Shift + K` | 新しいタスク |

### メール操作
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + R` | 返信 |
| `Ctrl + Shift + R` | 全員に返信 |
| `Ctrl + F` | 転送 |
| `Ctrl + M / F9` | 送受信 |
| `Ctrl + D` | 削除 |
| `Ctrl + Q` | 既読にする |
| `Ctrl + U` | 未読にする |
| `Ctrl + Shift + I` | 受信トレイへ移動 |
| `Ctrl + Shift + O` | 送信トレイへ移動 |

### メール作成
| ショートカット | 動作 |
|--------------|------|
| `Alt + S` または `Ctrl + Enter` | 送信 |
| `Ctrl + K` | 名前のチェック（あて名解決） |
| `Ctrl + Shift + C` | Cc欄を表示 |
| `Ctrl + Shift + B` | Bcc欄を表示 |
| `F7` | スペルチェック |

### フォルダー移動
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + 1` | メール表示 |
| `Ctrl + 2` | 予定表 |
| `Ctrl + 3` | 連絡先 |
| `Ctrl + 4` | タスク |
| `Ctrl + 5` | メモ |
| `Ctrl + Y` | フォルダーへ移動（名前指定） |

### 予定表
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + G` | 日付ジャンプ |
| `Alt + ←/→` | 前日/翌日へ |
| `Ctrl + Shift + 2/3/4` | 月/週/日表示切替 |

### 検索
| ショートカット | 動作 |
|--------------|------|
| `Ctrl + E` | 検索ボックスにフォーカス |
| `Ctrl + Shift + F` | 詳細検索 |
| `Ctrl + Shift + J` | 迷惑メールとして報告 |

## 🖥 Windows

### 基本操作
| ショートカット | 動作 |
|--------------|------|
| `Win` | スタートメニュー |
| `Win + D` | デスクトップ表示/復元 |
| `Win + E` | エクスプローラー |
| `Win + I` | 設定 |
| `Win + R` | ファイル名を指定して実行 |
| `Win + L` | PCをロック |
| `Win + X` | クイックリンクメニュー |
| `Win + A` | クイック設定 |

### ウィンドウ操作
| ショートカット | 動作 |
|--------------|------|
| `Win + →` | 右半分にスナップ |
| `Win + ←` | 左半分にスナップ |
| `Win + ↑` | 最大化 |
| `Win + ↓` | 最小化/元に戻す |
| `Win + Shift + →/←` | 別モニターに移動 |
| `Alt + F4` | アプリケーションを閉じる |
| `Win + M` | すべて最小化 |
| `Win + Shift + M` | 最小化を元に戻す |

### 仮想デスクトップ
| ショートカット | 動作 |
|--------------|------|
| `Win + Tab` | タスクビュー |
| `Win + Ctrl + D` | 新しいデスクトップ作成 |
| `Win + Ctrl + F4` | 現在のデスクトップを閉じる |
| `Win + Ctrl + →/←` | デスクトップ切替 |

### スクリーンショット・録画
| ショートカット | 動作 |
|--------------|------|
| `Win + Shift + S` | スクリーンショット（Snipping Tool） |
| `Win + PrtScn` | 画面全体を保存 |
| `PrtScn` | 画面全体をクリップボードへ |
| `Alt + PrtScn` | アクティブウィンドウをクリップボードへ |
| `Win + G` | ゲームバー（録画） |
| `Win + Alt + R` | 画面録画の開始/停止 |

### タスクバー
| ショートカット | 動作 |
|--------------|------|
| `Win + 1〜9` | タスクバーのN番目を起動/切替 |
| `Win + T` | タスクバーをフォーカス |
| `Win + S` | 検索（Windows Search） |

### クリップボード・入力
| ショートカット | 動作 |
|--------------|------|
| `Win + V` | クリップボード履歴 |
| `Win + .` （ドット） | 絵文字パネル |
| `Win + H` | 音声入力 |
| `Win + Space` | 入力言語の切り替え |

### ユーティリティ
| ショートカット | 動作 |
|--------------|------|
| `Win + P` | マルチディスプレイ切替 |
| `Win + K` | ワイヤレスディスプレイ接続 |
| `Win + +` | 拡大鏡（ズームイン） |
| `Ctrl + Shift + Esc` | タスクマネージャー |
| `Win + Pause` | システム情報 |

### ファイル操作（エクスプローラー）
| ショートカット | 動作 |
|--------------|------|
| `F2` | 名前の変更 |
| `F5` | 更新 |
| `Ctrl + Shift + N` | 新しいフォルダー作成 |
| `Alt + ↑` | 親フォルダーへ移動 |
| `Alt + ←/→` | 戻る/進む |
| `Shift + Delete` | 完全に削除 |

### アクセシビリティ
| ショートカット | 動作 |
|--------------|------|
| `Win + U` | アクセシビリティ設定 |
| `Win + Enter` | ナレーター切替 |
| `Win + Ctrl + C` | カラーフィルター切替 |

---

以上、**Excel** 10カテゴリ・**Outlook** 6カテゴリ・**Windows** 9カテゴリ の全ショートカットです。
